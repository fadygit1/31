CONSTRUCTION MANAGEMENT SYSTEM
================================

QUICK START (English):
1. Double-click "run.bat"
2. Wait for browser to open
3. Start using the system!

TROUBLESHOOTING:
- If run.bat doesn't work, try "install.bat" first
- If installation fails, run "check.bat" to diagnose
- Make sure Node.js is installed from nodejs.org

REQUIREMENTS:
- Windows 7 or newer
- Node.js (download from nodejs.org)
- 2GB RAM minimum
- Internet connection (first time only)

FILES EXPLANATION:
- run.bat: Main launcher (recommended)
- install.bat: Install/reinstall dependencies
- check.bat: System diagnostics
- quick.bat: Quick start without checks
- shortcut.bat: Create desktop shortcut

The system automatically detects project structure:
- Looks for src/main.tsx in current directory
- Falls back to project/src/main.tsx if not found
